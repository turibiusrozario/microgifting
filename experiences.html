2026 - Spring

The concept of Micro Gifting was first materialized on January 9th, 2026. A detailed document was created to list the Purpose, Scope, Monetary Logistics, Gift Logistics, Scale Up, and Wish List. This website was first created on January 15, 2026. GitHub pages was utilized for deployment. The theme for the website is 'Spectral' by HTML5UP; you can view their website here. The domain was purchased via PorkBun with default configurations for Github. The price of the domain was $10.81 paid by Turibius. Auto-renewal was set to off (please remind Turibius to auto-renew in January of next year just in case he forgets!)

Particular note from Turibius: Read the Purpose! It is very easy to start brainstorming different ideas, solutions, and strategies, and eventually deviate from the core values that micro gifting started off with. Reading the Purpose simultaneously aided me a lot in defining clear goals and success criteria.
